Python → Java Cheat Sheet

Core data structures for LeetCode & interviews

1Dynamic Array

Python
nums = [1, 2, 3]
nums.append(4)
Java
List<Integer> nums = new ArrayList<>();
nums.add(1);
nums.add(2);
  • ArrayList is the closest equivalent to Python's list.
  • Supports dynamic resizing.

2Stack

Python
stack = []
stack.append(1)
top = stack.pop()
Java (recommended)
Deque<Integer> stack = new ArrayDeque<>();
stack.push(1);
int top = stack.pop();
Alternative (not recommended)
Stack<Integer> stack = new Stack<>();
  • Most devs prefer Deque/ArrayDeque over Stack.
  • push() adds to top; pop() removes & returns top.

3Queue

Python
from collections import deque
q = deque()
q.append(1)
x = q.popleft()
Java
Queue<Integer> q = new ArrayDeque<>();
q.offer(1);
int x = q.poll();
  • offer() inserts; poll() removes from the front.
  • ArrayDeque is generally preferred over LinkedList.

4Hash Map

Python
freq = {}
freq[1] = 5
Java
Map<Integer, Integer> freq = new HashMap<>();
freq.put(1, 5);

5Hash Set

Python
seen = set()
seen.add(1)
Java
Set<Integer> seen = new HashSet<>();
seen.add(1);

6Min Heap

Python
import heapq
heap = []
heapq.heappush(heap, 5)
smallest = heapq.heappop(heap)
Java
PriorityQueue<Integer> heap =
    new PriorityQueue<>();
heap.offer(5);
int smallest = heap.poll();
  • Java's PriorityQueue is a min heap by default.
